# Unselected Architecture

The unselected architecture for this submission package is:

- `Blackboard Architecture`

This folder contains:

- `frontend/`
  The shared React frontend used in the demo
- `backend/Unselected/`
  The blackboard backend implementation

Notes for the grader:

- install frontend dependencies from `Unselected/frontend/`
- install backend dependencies from `backend/Unselected/requirements.txt`
- run the backend with `python -m uvicorn app.main:app --reload --port 8001`
- this package is included because the course guideline requires both candidate architectures to remain available

Current honest note:

- the active development repo now uses `backend/Selected/` and `backend/Unselected/`
