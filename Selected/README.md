# Selected Architecture

The selected architecture for this submission package is:

- `Layered Architecture`

This folder contains:

- `frontend/`
  The shared React frontend used in the demo
- `backend/Selected/`
  The layered backend implementation

Notes for the grader:

- start this package the same way as the main project README describes
- install frontend dependencies from `Selected/frontend/`
- install backend dependencies from `backend/Selected/requirements.txt`
- run the backend with `python -m uvicorn app.main:app --reload --port 8000`

Current honest note:

- the active development repo now uses `backend/Selected/` and `backend/Unselected/`
- this folder remains a submission-oriented reference note with `layered` marked as the selected candidate
