# CS5319 Final Project - AI Healthcare Chatbot

This project implements an AI Healthcare Chatbot using two different software architectures for comparison:

- **Layered Architecture** - Traditional n-tier architecture
- **Blackboard Architecture** - Pattern-based architecture for complex problem solving

## Project Structure

```
CS5319-FINAL-PROJECT
├── frontend/          # React frontend application
├── backend/
│   ├── layered/       # Layered architecture implementation
│   └── blackboard/    # Blackboard architecture implementation
├── docs/              # Documentation, diagrams, and screenshots
└── shared/            # Shared data and prompts
```

## Quick Start
